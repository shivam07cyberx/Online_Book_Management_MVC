# 📚 Online Book Management System

A Java-based **Online Book Management System** developed using **Spring MVC, JSP, Spring Data JPA, Hibernate, and MySQL**. This application provides complete CRUD functionality with additional features like pagination, sorting, searching, and efficient book deletion.

## 🚀 Features

- ➕ Add a new book
- 📖 View all books
- ✏️ Update book details
- ❌ Delete books efficiently
- 🔍 Search book by ID
- 📄 Pagination (Displays 5 books per page)
- 🔃 Sorting of books
- 🎨 User-friendly JSP interface
- ⚡ MVC Architecture
- 🗄️ Database integration using MySQL

## ⚙️ Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/your-username/online-book-management.git
```

2. Open the project in Eclipse or IntelliJ IDEA.

3. Configure the MySQL database.

4. Update database credentials in the configuration file.

5. Run the project on Apache Tomcat.

6. Open the application in your browser.
